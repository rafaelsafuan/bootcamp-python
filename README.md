# bootcamp-python

Aprendizado de Python aplicado ao mercado baseado em exercícios de Python com níveis ascendentes, organizado em semanas e módulos progressivos.

## Objetivo

Aprender Python de um jeito prático e útil, com exercícios aplicado ao mercado. Sou iniciante no mundo tech, com ampla bagagem comercial e com a intenção de suplementar o meu perfil com fundamentos de programação. Acredito que quem não se reinventar nesse mundo direcionado pela tecnologia vai ficar obsoleto a medio / longo prazo. Minha ideia é realmente aprender e não só ter um certificado ou diploma.
